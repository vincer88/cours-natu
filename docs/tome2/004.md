# 4) Qui est élu lors des élections municipales ?

## Réponse courte

Lors des élections municipales, les électeurs élisent **les conseillers municipaux** (la liste / le conseil municipal). Ensuite, **le maire est élu par le conseil municipal** parmi ses membres.

## Réponse détaillée (mini‑chapitre didactique)

### 1) Introduction

Beaucoup de personnes répondent spontanément : « on élit le maire ». Ce n’est pas totalement faux dans l’esprit, mais juridiquement et institutionnellement, ce n’est pas la bonne réponse attendue. En France, aux municipales, on élit d’abord le **conseil municipal** ; puis le conseil élit le **maire**.

### 2) Définitions et périmètre

- **Commune** : collectivité territoriale de base (mairie).
- **Conseil municipal / conseillers municipaux** : assemblée élue qui gère la commune.
- **Maire** : exécutif de la commune ; il préside le conseil municipal et applique ses décisions.

### 3) Contexte institutionnel

La commune est un échelon historique de l’organisation territoriale française. Les municipales sont donc un scrutin essentiel du quotidien : urbanisme, écoles primaires, état civil, action sociale locale, etc. La logique est représentative : les habitants élisent une équipe (des conseillers) qui va ensuite choisir un maire.

### 4) Fonctionnement : déroulé après le vote

1. Les électeurs votent aux **élections municipales**.
2. Le résultat détermine la composition du **conseil municipal** (conseillers élus).
3. Lors de la première réunion (le “conseil d’installation”), les conseillers municipaux élisent :
   - **le maire**,
   - puis les adjoints.

**Relances d’entretien possibles**
- *« Donc le maire n’est jamais élu par les citoyens ? »* → Il n’est pas élu **directement** : il est élu **par le conseil**, mais les citoyens choisissent les conseillers qui le choisiront.
- *« Quelle est la différence entre un maire et un conseiller municipal ? »* → Le conseiller est membre de l’assemblée ; le maire est l’exécutif et représente la commune.

### 5) Exemple concret 1

Dans une ville, une liste obtient la majorité des sièges au conseil municipal. Une semaine plus tard, les conseillers se réunissent. Ils votent : la tête de liste devient souvent maire, mais ce n’est pas automatique juridiquement : c’est le vote des conseillers qui le décide.

### 6) Exemple concret 2

Dans une commune, la liste arrivée en tête n’a pas une majorité stable. Au moment d’élire le maire, une alliance entre plusieurs groupes de conseillers se forme et élit un maire différent de la tête de liste initiale. Cela illustre que le maire dépend du **conseil**.

### 7) Limites et confusions fréquentes

- **Confusion 1 : “on élit le maire”** → réponse incomplète : on élit **les conseillers municipaux**.
- **Confusion 2 : “le conseil municipal = la mairie”** → la mairie est un lieu/une administration ; le conseil est une **assemblée élue**.
- **Nuance** : les règles de scrutin varient selon la taille des communes (liste, panachage dans certaines petites communes), mais l’idée centrale reste : **élection du conseil**, puis **élection du maire par le conseil**.

### 8) Synthèse

Réponse attendue :
- on élit **les conseillers municipaux**,
- puis le conseil municipal élit **le maire**.

**Réponse modèle (entretien)** : *« Aux municipales, on élit les conseillers municipaux. Ensuite, le conseil municipal élit le maire parmi ses membres. »*

## QCM d’entraînement

**Question :** Lors des élections municipales, que choisissent les électeurs ?

A. Le maire directement.
B. Les conseillers municipaux. ✅
C. Le préfet.
D. Les sénateurs.

## Pièges à éviter

- Répondre uniquement “le maire”.
- Oublier l’étape : **le conseil municipal élit le maire**.
