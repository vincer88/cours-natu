# 1) Parmi ces textes, lequel a été adopté sous Napoléon Ier ?

## Réponse courte

Le texte emblématique adopté sous Napoléon Ier est le **Code civil** (souvent appelé « Code Napoléon », publié en **1804**).

## Réponse détaillée (mini‑chapitre didactique)

### 1) Introduction

À l’entretien, cette question sert à vérifier des repères historiques : Napoléon Ier (Premier Empire) est associé à une grande œuvre de codification. La réponse la plus attendue est : **le Code civil de 1804**.

### 2) Définitions et périmètre

- **Napoléon Ier** : Empereur des Français, période 1804‑1814 (et 1815 pour les Cent‑Jours).
- **Code civil** : texte majeur qui organise les règles de la vie civile (personnes, famille, propriété, contrats…).
- **Codification** : regroupement et organisation des règles juridiques dans un code.

### 3) Contexte historique et institutionnel

Après la Révolution, la France cherche à stabiliser et unifier le droit sur tout le territoire. Sous Napoléon, l’État renforce son organisation et produit des codes structurants. Le Code civil devient une référence durable du droit français.

### 4) Fonction : pourquoi c’est important

Le Code civil fixe des règles communes pour :
- la famille et l’état des personnes,
- la propriété,
- les contrats et obligations,
- la transmission des biens.

Il apporte une cohérence nationale et une sécurité juridique.

**Relances d’entretien possibles (et réponses modèles)**
- *« Pourquoi on l’appelle Code Napoléon ? »* → *« Parce qu’il a été promulgué sous Napoléon et associé à sa politique de codification. »*
- *« Il date de quand ? »* → *« 1804. »*

### 5) Exemple concret 1

Quand on signe un contrat (location, achat…), beaucoup de principes généraux (obligations, responsabilité) relèvent du droit civil organisé par le Code civil.

### 6) Exemple concret 2

Les règles de propriété et de transmission (héritage) s’appuient sur des principes du droit civil codifié.

### 7) Limites et confusions fréquentes

- **Confusion 1 : mélanger Code civil et Constitution.** Le Code civil est un code de droit privé ; la Constitution organise les pouvoirs publics.
- **Confusion 2 : croire que le Code civil n’a jamais changé.** Il a été très réformé au fil du temps.
- **Nuance** : Napoléon est associé à plusieurs codes, mais le **Code civil** est le plus célèbre.

### 8) Synthèse

Texte adopté sous Napoléon Ier (repère classique) : **Code civil (1804)**.

**Réponse modèle (entretien)** : *« Le Code civil (1804), appelé Code Napoléon. »*
